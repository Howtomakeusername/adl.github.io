Modeling an AI
  • - Using Roboflow, I can easily find premade datasets (Kaggle is also good for this) or customize my own ones. However, I only use it to create bounding boxes as it requires premium subcription to do anything else
  • - For simple object detection AI, I can go on Ultralytics and using the Yolov8s or Yolov8m algorithm as a base to create my custom model depending on my need
  • - For more complex ones (like fighting detection AI), I find Yolov8 a bit inaccurate without supplement code so I also had to learn using keypoints to detect movement
  • - Augmentation is really important and sometimes I have to tweak the value a little bit because the default setting of ultralytics just isn't enough and it leads to model being unable to detect anything in many situations
